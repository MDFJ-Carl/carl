# WIN32 Research Notes

## About Win32

Win32 is the main Windows API created by Microsoft for desktop application development on Microsoft Windows operating systems.

It provides access to:

- Window creation
- User input
- Graphics
- File management
- System-level functions

DirectX is also a Microsoft technology used for graphics, multimedia, and game development.

## Purpose Inside Carl

The Carl project studies Win32 and DirectX concepts for educational and research purposes.

Our goal is to understand how operating system APIs and graphical systems work in order to explore compatibility concepts and future system architecture ideas for Carl-OS.

## Important Notice

Carl does NOT copy proprietary Microsoft Windows source code.

We do NOT claim ownership of Win32 or DirectX technologies.

Microsoft is the original creator and owner of:

- Win32 API
- DirectX

Carl only researches publicly documented behavior and concepts.

## Research Goals

- Learn low-level application architecture
- Study graphical window systems
- Understand event/message systems
- Explore compatibility layer concepts
- Improve operating system development knowledge
